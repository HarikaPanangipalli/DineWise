app.db.database module¶

Database module for the DineWise application.

This module sets up a connection to MongoDB using the motor library and provides references to collections used throughout the application.

ivar db_client:

The MongoDB client instance.

vartype db_client:

AsyncIOMotorClient

ivar database:

The main MongoDB database instance.

vartype database:
